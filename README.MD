##### tabBarIcon

tabBar的icon, 可以接受一个函数, 函数的参数的三个属性为focused(是否聚焦), horizontal(横竖屏模型), tintColor(active的Color)

##### activeTintColor

激活的颜色

##### inactiveTintColor
